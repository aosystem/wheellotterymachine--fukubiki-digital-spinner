# Wheel Lottery Machine: Japanese Fukubiki Experience 🎡🎊

**Wheel Lottery Machine** is an interactive and highly customizable **Digital Raffle Tool** built with **Flutter**. It faithfully recreates the excitement of the traditional Japanese "Fukubiki" (rotating drum lottery), allowing users to manage physical-style ball counts and prize distributions for events, parties, and giveaways.

## 🌟 Key Features
- **Authentic Simulation:** Experience the thrill of a spinning drum and the visual satisfaction of a randomly selected colored ball.
- **Customizable Inventory:** Assign prizes to up to **8 different colors** and set the exact number of winning balls for each.
- **Real-time Tracking:** The app keeps track of the remaining ball count, with an easy "refill" function when the drum is empty.
- **Engaging Visuals:** Colorful animations and intuitive controls make it perfect for live streaming and public events.
- **Flexible Setup:** Simply leave unused color fields blank for smaller prize pools.
- **Global Support:** Fully localized in **50+ languages**, bringing Japanese-style fun to a global audience.

## 🛠 Technical Overview
- **Framework:** Flutter
- **Language:** Dart
- **Logic:** State-managed inventory system that handles random draws without replacement (until refill), combined with physics-inspired spin animations.

*Note: This repository contains the core logic and source code. Assets (images, sound effects, and localization files) are excluded to protect intellectual property.*
